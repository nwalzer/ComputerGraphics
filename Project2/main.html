Please use a browser that supports the "canvas" tag.

Press A to zoom out, Z to zoom in, Y to translate the drawing upwards, U to translate the drawing downwards

X to toggle translating to the right, C to translate to the left, R to toggle x-axis rotation, B to toggle pulsing, N to toggle displaying the normal vectors

Select from the mesh color picker to change the color of the mesh. Select from the normal color picker to change the color of the surface normals
